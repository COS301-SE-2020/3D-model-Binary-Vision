Welcome to 3D Binary

Welcome to our 3D Binary application where we use modern algorithms and techniques to preview your patient's teeth in a dynamic three-dimensional environment.

With our 3D Binary system you are able to, by making use of an intraoral camera, take several images of your patient's teeth which will then be used to determine and create an accurate depiction in a three-dimensional space which will all be done in real-time.

Each model of the patient's teeth is securely stored in an organised database along with the patient's information for easy access. Once the model is generated, it can easily be reopened and previewed at any time.
